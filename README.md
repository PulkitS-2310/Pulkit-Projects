# Pulkit Projects

This repository contains my personal, academic, research, and cybersecurity projects. I am currently studying Computer Science at Iowa State University with a minor in Cybersecurity Engineering. My interests include cybersecurity, software development, artificial intelligence, federated learning, and full-stack/mobile development.

## Projects

### Smart Honeypot + AI Attacker Profiling

Cybersecurity dashboard built with Python, Streamlit, Cowrie, and Pandas to parse honeypot logs, analyze attacker behavior, and display threat activity.

### PingMe Social Media App

Android social media application built with Java, Android Studio, Postman, and WebSockets. Focused on frontend development, media posting, user activity, and real-time notifications.

### Hospital Penetration Testing Project

Simulated hospital network penetration testing project using Kali Linux and security tools to identify exposed services, misconfigurations, exploitation paths, and remediation steps.

### Checkers AI Simulation Engine

Python-based Checkers simulation engine using rule-based gameplay, board-state modeling, Alpha-Beta pruning, and Monte Carlo Tree Search.

### FLBench Research

Research work focused on federated learning benchmarking, FedAvg, FedMeta, SplitFed, HeteroFL, federated learning heterogeneity, personalization, and scoring criteria for fair method comparison.
